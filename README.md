# Exam Revision — BPSC / UPSC Study Notes

Dark-themed, NCERT-based study notes for BPSC and UPSC preparation.

## Contents

- **History** — 8 topics (Indus Valley to European Powers)
- **Current Affairs** — Monthly compilations (Jan–Apr 2026)
- **Economics** — Budget 2026-27, Economic Survey 2025-26, 100+ Economic Terms
- **Geography** — Bihar district-wise rivers, soil, crops, minerals, forest cover

## Features

- Tailwind CSS dark theme with gradient cards
- Day-wise, category-wise current affairs
- District-wise coverage for geography
- MCQs and quick-fact cards in every topic
- Searchable 114-term economics glossary with mnemonics
